Wie breit ist der Fluss? 
DE = 35 m, FG = 25 m, DF = 21 m. 



Die Dreiecke DEC und FGC sind ähnlich, 
weil sie in 2 Winkeln übereinstimmen.  

                        FG      25 m      5 
Ähnlichkeitsfaktor k = ---- = -------- = ---    
                        DE      35 m      7 

CF = CD - FD |+FD 

CD = CF + FD 

     CF        CF  
k = ---- = ----------|*CF + FD 
     CD     CF + FD 

k * (CF + FD) = CF 

k * CF + k * FD = CF |-k * CF 

k * FD = CF * (1 - k) |:(1 - k) 

      k * FD      5/7 * 21 m     5 * 21 m 
CF = --------- = ------------ = ---------- = 52,5 m 
       1 - k       1 - 5/7          2    

oder Strahlensatz: 

 FG     CF 
---- = ----   
 DE     CD 

Über Kreuz multipliziert: 

FG * CD = CF * DE 

FG * (CF + FD) = CF * DE 

FG * CF + FG * FD = CF * DE |-FG * CF 

FG * FD = CF * (DE - FG)|:(DE - FG) 

       FG * FD      25 * 21 
CF = ----------- = --------- = 52,5 m 
       DE - FG      35 - 25 

oder  

25 : 35 = FC : (FC + 21) 

Inneres Produkt = äußeres Produkt 

FC * 35 = 25 * (FC + 21) 

FC * 35 = 25 * FC + 525 |-25FC 

10FC = 525 |:10 

FC = 52,5 m
