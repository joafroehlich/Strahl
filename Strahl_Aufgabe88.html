Aufgabe 88

Bestimmen Sie die Länge der Strecke x.  
Alle Maße in cm. BD und CE verlaufen parallel. 



Strahlensatz: 

 1,8 + x      3,6       
---------- = ----- |*1,8   
    1,8       2,7  

              3,6 cm * 1,8 cm 
1,8 cm + x = ------------------ |-1,8 cm 
                   2,7 cm 

     3,6 cm * 1,8 cm 
x = ------------------ - 1,8 cm 
          2,7 cm 

x = 0,6 cm 

oder  

(x + 1,8) : 1,8 = 3,6 : 2,7 

Inneres Produkt = äußeres Produkt 

3,6 * 1,8 = (x + 1,8) * 2,7 |:2,7 

 3,6 cm * 1,8 cm 
------------------ = x + 1,8 cm |-1,8 cm 
       2,7 cm 

     3,6 cm * 1,8 cm 
x = ------------------ - 1,8 cm 
          2,7 cm 

x = 0,6 cm
