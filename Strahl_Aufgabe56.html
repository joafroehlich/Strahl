Aufgabe 56

Ein Schiff peilt ein zweites senkrecht 
zu seiner Fahrtrichtung an. Der Abstand 
beträgt 3,6 km. Nach 8 km beträgt der 
Abstand noch 1,7 km. 
Nach wieviel weiteren km treffen sie sich? 



Die Dreiecke ADB und CDE sind ähnlich, 
weil sie in 2 Winkeln übereinstimmen.  

                         3,6 km      
Ähnlichkeitsfaktor k = -----------   
                         1,7 km 

     AD 
k = ----- |*x 
      x 

k * x = AD |:k 

     AD 
x = ---- 
     k 

AD = 8 km + x 

     8 + x     1,7 * (8 + x) 
x = ------- = ---------------- |*3,6 
      3,6           3,6     
     ----- 
      1,7   

3,6x = 13,6 + 1,7x |-1,7x 

1,9x = 13,6 |:1,9 

x = 7,16 km 

oder Strahlensatz: 

  3,6      1,7 
------- = ------    
 8 + x      x 

Über Kreuz multipliziert: 

3,6x = 13,6 + 1,7x |-1,7x 

1,9x = 13,6 |:1,9 

x = 7,16 km 

oder  

8 + x : x = 3,6 : 1,7 

Inneres Produkt = äußeres Produkt 

x * 3,6 = 1,7 * (8 + x)  

x * 3,6 = 13,6 + 1,7 * x |-1,7x 

1,9x = 13,6 |:1,9 

x = 7,16 km
