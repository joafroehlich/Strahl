Bestimmen Sie die Flussbreite CD, wenn 
die Strecken mit DE = 20 m, FG = 28 m 
und DF = 20 m abgesteckt wurden.  



DB||FG --> 

Strahlensatz:  

 CD     DE       
---- = ----  |*CF  
 CF     FG  

CF = CD + DF 

       (CD + DF) * DE 
CD = ------------------ |*FG 
              FG 

CD * FG = CD * DE + DF * DE |-CD * DE 

CD * (FG - DE) = DF * DE |:(FG - DE) 

       DF * DE      20 m * 20 m  
CD = ----------- = ------------- = 40 m 
       FG - DE      28 m - 20 m 

oder 

CD : CF = DE : FG 

Inneres Produkt = äußeres Produkt 

CF * DE = CD * FG 

(CD + DF) * DE = CD * FG 

CD * DE + DF * DE = CD * FG |-CD * DE 

DF * DE = CD * (FG - DE) 

      DF * DE      20 m * 20 m  
CD = ---------- = ------------- = 40 m 
      FG - DE      28 m - 20 m
