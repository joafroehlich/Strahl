Bestimmen Sie die Längen der 2 fehlenden Seiten, 
wenn BD||CE und a = 7,2 cm, c = 6,8 cm, d = 10,2 cm 
und e = 5,4 cm 



Strahlensatz: 

 b     d       
--- = --- |*a    
 a     c  

     d * a       10,2 cm * 7,2 cm        
b = -------- = -------------------- = 10,8 cm 
        c              6,8 cm 

oder  

b : a = d : c 

Inneres Produkt = äußeres Produkt 

a * d = b * c |:c 

     d * a       10,2 cm * 7,2 cm        
b = -------- = -------------------- = 10,8 cm 
        c              6,8 cm 

Strahlensatz: 

 f     d       
--- = --- |*e    
 e     c

oder 

f : e = d : c 

Inneres Produkt = äußeres Produkt 

e * d = f * c |:c
