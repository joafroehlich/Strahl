Bestimmen Sie die Länge der unzugänglichen 
Strecke CD. 
ED = 324 m, CF = 129,6 m, EG = 86,4 m 



Die Dreiecke CDF und EDG sind ähnlich, 
weil sie in 2 Winkeln übereinstimmen.  

                         CF      129,6 m 
Ähnlichkeitsfaktor k = ----- = ----------- = 1,5    
                         EG      86,4 m 

     CD 
k = ----- |*ED 
     ED 

CD = k * ED = 1,5 * 324 m = 486 m 

oder Strahlensatz: 

 CF     CD 
---- = ----- |*ED  
 EG     ED 

      129,6 
CD = ------- * 324 m = 486 m 
      86,4 

oder  

129,6 : 86,4 = CD : 324 

Inneres Produkt = äußeres Produkt 

CD * 86,4 = 129,6 * 324 |:86,4 

CD = 486 m
