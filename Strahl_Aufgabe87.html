Bestimmen Sie die Länge der Strecke x.  
Alle Maße in cm. BD und CE verlaufen parallel. 



Strahlensatz: 

     x       5,2       
--------- = -----    
 x + 2,2     7,8 

Über Kreuz multipliziert: 

x * 7,8 = (x + 2,2) * 5,2 |-5,2x 

x * 2,6 = 11,44 |:2,6 

x = 4,4 cm 

oder  

x : (x + 2,2) = 5,2 : 7,8 

Inneres Produkt = äußeres Produkt 

(x + 2,2) * 5,2 = x * 7,8 |-5,2x 

x * 2,6 = 11,44 |:2,6 

x = 4,4 cm
